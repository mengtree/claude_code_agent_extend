# Scheduler Skill - 使用说明

## 功能概述

Scheduler Skill 用于创建和管理定时任务，支持：
- ✅ 创建一次性任务
- ✅ 创建延迟任务（N分钟后）
- ✅ 创建周期性任务（cron表达式）
- ✅ 查询所有任务
- ✅ 查看任务详情
- ✅ 取消定时任务

## 命令使用

### 1. 列出所有任务

```bash
cd .claude/skills/scheduler/scripts
python main.py list
```

输出示例：
```
📋 共有 3 个定时任务

1. 🔄 每2分钟活动提醒
   ID: 832ad502 | 状态: 🟢 active
   下次执行: 2026-03-11T09:26:00.000+08:00

2. 📅 买菜提醒
   ID: dc357439 | 状态: 🟢 active
   下次执行: 2026-03-11T09:15:44.000+08:00
```

### 2. 查看任务详情

```bash
python main.py get --id 832ad502
```

可以只输入任务ID的前几位，系统会自动匹配。

### 3. 取消任务

```bash
python main.py cancel --id 832ad502
```

取消时会显示确认信息。

## 任务文件命名规则

任务文件名格式：`{scheduleId}.json`

- `scheduleId`: UUID格式的任务ID（全局唯一）

示例：
- `832ad502-4821-4ca3-afe4-bc6e4ab7d077.json`
- `dc357439-6c9a-441d-a8b5-7856db6a6e51.json`

## 任务类型说明

| 类型 | 说明 | 图标 |
|------|------|------|
| one_time | 一次性任务 | 📅 |
| delay | 延迟任务 | ⏰ |
| cron | 周期性任务 | 🔄 |

## 常见问题

### Q: 如何知道某个任务的具体内容？
A: 使用 `python main.py get --id <任务ID>` 查看详情。

### Q: 取消任务后会怎样？
A: 任务文件会被删除，任务不再执行。已执行的周期任务历史记录不受影响。

### Q: 如何修改已有任务？
A: 目前不支持直接修改。建议先取消旧任务，再创建新任务。
